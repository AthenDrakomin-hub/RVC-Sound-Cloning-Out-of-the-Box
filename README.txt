RVC 音色克隆整合版
===============

这是一个整合了Web界面和核心应用程序的简化版本，旨在提供更加直观和易用的用户体验。

主要改进：
---------
1. 整合了Web界面和应用程序，避免用户混淆
2. 提供清晰的功能选项菜单
3. 简化了使用流程，只需要三步：
   - 准备音频文件
   - 启动应用程序
   - 选择所需功能
   
目录说明：
--------
- assets/: 预训练模型和权重文件
- configs/: 配置文件
- infer/: 核心音色克隆算法模块
- gui_v1.py: 实时音色克隆通话界面
- infer-web.py: Web训练和文件处理界面
- simple_rvc.py: 纯命令行版本
- 启动应用程序.bat: 主启动脚本
- 使用说明.txt: 详细使用指南

使用方法：
--------
双击运行 "启动应用程序.bat"，然后根据菜单选择所需功能：
1. 实时音色克隆通话（推荐）
2. 音频文件音色克隆
3. 训练新音色模型
4. 退出程序

注意事项：
--------
- 首次使用前请确保已安装所有依赖库
- 如遇到问题，请参考使用说明.txt 或联系技术支持
