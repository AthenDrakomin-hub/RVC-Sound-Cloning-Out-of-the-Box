RVC 音色克隆开箱即用包
====================

这是一个简化版的RVC音色克隆应用程序包，包含了运行RVC所需的核心文件。

包内容：
- infer-web.py: Web界面主程序
- gui_v1.py: 实时语音转换程序
- assets/: 预训练模型和资源文件
- configs/: 配置文件
- i18n/: 国际化语言文件
- infer/: 核心推理模块
- tools/: 工具脚本
- 启动RVC.cmd: 主启动器

使用方法：
1. 双击"启动RVC.cmd"
2. 根据菜单选择功能

注意：首次使用可能需要安装依赖：
pip install -r requirements.txt

更多信息请查看"使用说明.txt"文件。
