# RVC Sound Cloning Out-of-the-Box

## 简介

这是一个开箱即用的RVC（Retrieval-based Voice Conversion）音色克隆应用程序包。它包含了运行RVC音色克隆所需的所有文件和依赖，用户可以直接使用而无需复杂的环境配置。

## 特点

- 🎯 **开箱即用**：解压后即可直接运行，无需额外配置
- 🖥️ **双界面支持**：包含Web训练界面和实时语音转换界面
- 📦 **一体化解决方案**：包含所有依赖库和预训练模型
- 🇨🇳 **中文友好**：完整的中文界面和说明文档
- 🚀 **易于使用**：简单的启动器和清晰的操作指南

## 系统要求

- Windows 10/11 64位操作系统
- 至少8GB内存
- 推荐有GPU支持以获得更好的性能

## 使用方法

1. 下载并解压本仓库
2. 双击运行 `启动RVC.cmd`
3. 根据菜单选择功能：
   - 选项1：启动Web训练界面（用于训练音色模型）
   - 选项2：启动实时语音转换（用于实时变声）
   - 选项3：退出程序

## 功能说明

### Web训练界面
- 可视化训练音色模型
- 支持一键训练
- 提供模型推理功能
- 访问地址：http://localhost:7897

### 实时语音转换
- 实时语音变声
- 支持麦克风输入和扬声器输出
- 低延迟音频处理

## 音色训练流程

1. 准备音频素材：
   - 收集3-30分钟清晰的语音音频
   - 支持WAV、MP3、FLAC等格式
   - 将音频文件放入 `datasets/` 目录

2. 启动Web训练界面

3. 在训练标签页配置参数：
   - 实验名称：自定义名称
   - 采样率：40k（语音）或48k（音乐）
   - 音高指导：建议选择"是"
   - 版本：建议使用"v2"

4. 设置训练文件夹路径

5. 点击"一键训练"按钮

6. 训练完成后在推理界面使用模型

## 注意事项

1. 确保音频文件质量良好，无杂音
2. 训练过程需要一定时间，请耐心等待
3. 训练结果保存在 `logs/` 目录中
4. 生成的模型文件可用于实时语音转换
5. 如遇到问题，请查看控制台输出的错误信息
