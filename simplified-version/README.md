# RVC 音色克隆简化版

## 简介

这是一个专为普通用户设计的RVC（Retrieval-based Voice Conversion）音色克隆简化版包。它去除了复杂的环境配置步骤，让用户能够快速上手使用音色克隆功能。

## 特点

- 🎯 **零配置启动**：无需复杂的环境配置，双击即可使用
- 🚀 **自动依赖检查**：自动检测并安装缺失的依赖库
- 🎵 **双模式支持**：支持Web训练界面和实时语音转换
- 📋 **详细文档**：包含使用说明和常见问题解答
- 🎨 **用户友好**：简洁明了的操作界面

## 系统要求

- **操作系统**：Windows 10/11 64位
- **内存**：至少8GB RAM
- **存储空间**：至少5GB可用空间
- **Python**：3.10或更高版本（启动器会自动检查）

## 使用方法

### 1. 首次使用

1. 安装Python 3.10或更高版本（如尚未安装）
   - 访问 https://www.python.org/downloads/
   - 下载并安装Python
   - 安装时务必勾选"Add Python to PATH"
   - 安装完成后重启计算机

2. 双击运行 `启动RVC.bat`

3. 程序会自动检查环境和依赖：
   - 如果缺少依赖库，程序会询问是否自动安装
   - 输入"Y"确认安装

### 2. 选择功能

- 输入"1"启动Web训练界面
- 输入"2"启动实时语音转换
- 输入"3"退出程序

### 3. Web训练界面使用

1. 启动后在浏览器中打开 http://localhost:7897
2. 训练新音色：
   - 点击"训练"标签页
   - 填写实验名称
   - 选择目标采样率（推荐40k）
   - 选择是否带音高指导（推荐是）
   - 填写训练文件夹路径
   - 点击"一键训练"

3. 使用训练好的音色：
   - 点击"推理"标签页
   - 选择训练好的模型
   - 上传音频文件进行转换

### 4. 实时语音转换使用

1. 启动后选择选项2
2. 在界面中：
   - 选择训练好的模型文件
   - 配置音频输入设备（麦克风）
   - 配置音频输出设备（扬声器）
   - 调整相关参数
3. 点击"开始音频转换"按钮开始实时变声

## 文件说明

- `使用说明.txt`：详细使用指南

- `infer-web.py`：Web训练界面主程序
- `gui_v1.py`：实时语音转换主程序

## 注意事项

1. 首次运行可能需要较长时间安装依赖库
2. 训练音色需要准备3-30分钟的清晰音频素材
3. 支持的音频格式：WAV、MP3、FLAC等
4. 训练好的模型会保存在logs文件夹中
5. 如遇到问题，请查看"常见问题FAQ.md"
