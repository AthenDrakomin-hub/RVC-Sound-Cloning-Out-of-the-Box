RVC 音色克隆整合版 - 版本说明
============================

版本目标：
--------
解决用户提出的"为什么要分开web和应用程序，能不能全部转换成应用程序，然后把web清理"的问题。

完成情况：
--------
✅ 已创建整合版本目录 (integrated-version)
✅ 已整合Web界面和核心应用程序
✅ 已提供清晰的功能选项菜单
✅ 已实现一键清理Web组件功能
✅ 已创建简化的使用流程

目录结构：
--------
integrated-version/
├── assets/                  # 预训练模型和权重文件
├── configs/                 # 配置文件
├── infer/                   # 核心音色克隆算法模块
├── gui_v1.py               # 实时音色克隆通话界面
├── infer-web.py            # Web训练和文件处理界面
├── simple_rvc.py           # 纯命令行版本
├── requirements.txt        # 依赖库列表
├── 启动应用程序.bat        # 主启动脚本（带清理选项）
├── 清理Web组件.bat         # 独立的清理工具
├── 使用说明.txt            # 详细使用指南
├── README.txt             # 简要说明
└── 版本说明.txt            # 本文件

核心功能：
--------
1. 实时音色克隆通话 - 克隆您的音色并在实时通话中使用
2. 音频文件音色克隆 - 将音色应用到音频文件上
3. 训练新音色模型 - 使用自己的数据训练专属模型

使用方法：
--------
1. 双击运行 "启动应用程序.bat"
2. 根据需要选择是否清理Web组件
3. 选择所需功能并按照提示操作

清理Web组件：
------------
如果只需要核心的实时音色克隆功能，可以选择清理Web组件：
1. 在启动应用程序时选择清理选项
2. 或者单独运行 "清理Web组件.bat"

优势特点：
--------
1. 整合了Web界面和应用程序，避免用户混淆
2. 提供清晰的功能选项菜单
3. 简化了使用流程，降低使用门槛
4. 保留了完整的功能，同时提供精简选项
5. 提供多种使用方式满足不同用户需求

注意事项：
--------
- 首次使用前请确保已安装所有依赖库
- 实时转换需要麦克风输入权限
- 训练过程可能需要较长时间，请耐心等待
